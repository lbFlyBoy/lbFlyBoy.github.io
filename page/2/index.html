python基础语法（一）

注释

单行注释

用#号、三个单引号或者三个双引号加代码来注释
如：# 第一步
   print '第一步'
   # 第二步
   print '第二步'

多行注释

’‘’ 这里是注释 '''
""" 这里是注释 """

特殊注释

# encoding = utf-8（等同于：# _*_ coding:utf-8 _*_)
python2 中不支持项目代码中注释中文

变量

变量定义方法很简单（跟其他编程语言比较起来来看真滴太simple）

x = 5
str = "abc"

数据类型

  • Numbers (数值类型）
  • Bool （Ture/False)
  • String
  • List
  • Set
  • Tuple (元组）
  • Dictory
  • NoneType (空类型)

查看数据类型方法：

type(数据)

类型转换

python为强类型语言，不同类型的数据不能参与计算，所以会涉及到类型转换，例如：

a = 3
b = '4'
c = a + b
运行的话会报错，因为变量a、b的数据类型不一致，需要转换成数据类型一致
c = a + int(b) 或者 c = str(a) + b

运算符

幂运算: 3 的 10 次方

3 ** 10

除法: 5 除以 2

除法 5 / 2 = 2.5
整除 5 // 2 = 2
余数 5 % 2 = 1

比较运算符：10 大于 5 小于 20 如果其他编程语言形式为：

num = 10
其他编程语言  num > 5 && num < 20
python语言   5 < num < 20

输入

  eval()      函数就是把接受到的结果当做代码来处理

  例如：
  result = eval("1+1")
  reslut = 2
  所以input可以看做是raw_input 和 eval综合的结果

python3
