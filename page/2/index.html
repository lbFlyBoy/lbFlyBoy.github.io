（当然除不带修饰符var的变量)

函数中定义变量提现声明的问题：

func();
var name = 'ls';
function  func() {
    console.log(name);
    var name = "zs";
}

答案是undefined
这个怎么理解那，首先程序会在函数中找name变量，如果有会打印 
函数内定义的变量，而变量的定义在打印之前，程序会将变量的声明     
放到打印之前定义放到打印之后，所以答案你懂得

类型转换

python为强类型语言，不同类型的数据不能参与计算，所以会涉及到类型转换，例如：

a = 3
b = '4'
c = a + b
运行的话会报错，因为变量a、b的数据类型不一致，需要转换成数据类型一致
c = a + int(b) 或者 c = str(a) + b

运算符

幂运算: 3 的 10 次方

3 ** 10

除法: 5 除以 2

除法 5 / 2 = 2.5
整除 5 // 2 = 2
余数 5 % 2 = 1

比较运算符：10 大于 5 小于 20 如果其他编程语言形式为：

num = 10
其他编程语言  num > 5 && num < 20
python语言   5 < num < 20
