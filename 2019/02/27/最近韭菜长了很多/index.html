最近韭菜长了很多

By dreamlee

发表于 2019-02-27

文章目录

最近股市行情不错，自媒体文章、朋友圈、生活的周围大家都在讨论着股票，这两天也接到过不少股票推销电话。行情好的时候大家都觉得自己是大神，也许有些人确实在股票中赚了点钱，这个时候也是我们这些小白韭菜长得最多的时候，我们不是大神，只是最近的行情好，只是运气好而已，大家记住股票的原则 “7赔2平1赚” 。

其实在上大学的时候我就做过小白韭菜，进入股市的原因就是看到周围同学炒股有赚钱的，自己也想着赚钱（没想过会赔，相信自己也是股神，哈哈），看着某个股票涨停好像就就是自己的股票，看着股票k线图的走势，觉得自己在最低点不就 OK 了 （好像自己一定能买到低点），可惜现实情况低点我们是买不到，却都买在了走势的凸点，贪婪的心理害死人，这就是我们当成韭菜的原因。

上大学也没什么钱，竟然想到了要缴纳的学费，因为学费只要在毕业前全部缴纳完成就可以，一万多块学费就投进去了，前期想着只要小心翼翼，走短线路线，涨一丢丢就卖掉的心理去买股票，第一天买股票竟然涨了还是涨停，第二天就卖掉了，心理想着自己挺厉害，股票这么容易就赚到钱。其实只不过是运气好而已。接下来换股炒短线，心理却发生了变化，股票涨了不想卖了，想着它再涨点再卖，结果拉高后跌停，这时候更不会卖了，还想着它会涨回来那，这就是人性的贪婪。后来整个股市行情也不好，快毕业了要交学费才忍痛 “割肉”。

其实我也不反对大家炒股，如果手里有点闲钱，也可以在行情好的时候放里面，权当学习了，但不建议大家做短线，对心理是个考验，当然更赚不到什么钱。
