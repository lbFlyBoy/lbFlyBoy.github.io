变量作用域问题

js不同与java、c语言特点：

{
    var name = 'zs';
}
console.log(name);

代码块中的变量js是可以访问的，也就是说js中只有函数中定义    
的变量是局部变量，其他地方是全局变量    
（当然除不带修饰符var的变量)

函数中定义变量提现声明的问题：

func();
var name = 'ls';
function  func() {
    console.log(name);
    var name = "zs";
}

答案是undefined
这个怎么理解那，首先程序会在函数中找name变量，如果有会打印 
函数内定义的变量，而变量的定义在打印之前，程序会将变量的声明     
放到打印之前定义放到打印之后，所以答案你懂得

自定义对象

通过构造函数自定义对象

function Person(name, age, gender) {
    this.name = name;
    this.age = age;
    this.gender = gender;
    this.running = function() {
      console.log(this.name + "is running");
    };
}
var person = new Person("xm", 19, "男");
person.running();

给对象添加属性和方法（其他编译性语言实现起来比较麻烦)

var person = {
        name:"llz",
        age:18
    };
// 添加属性height 方法sayHi、sleep     
person.height = 1.8;
// 点语法添加方法sayHi
person.sayHi = function () {
    console.log(person.name + "sayHi I's height :" + person.height);
};
// [字符串]添加方法sayHi
person["sleep"] = function () {
    console.log(person.age + "sayHi I's height :" + person.height);
};
